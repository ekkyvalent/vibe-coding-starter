# Starter Prompt: Personal Portfolio

Copy and paste this into Claude Code to build a personal portfolio or profile page.

---

## The prompt

```
Build me a personal portfolio page. Here's my info:

NAME: [your name]
ROLE: [your title or what you do, e.g. "Product Designer" or "Indie Hacker"]
BIO: [2-3 sentences about yourself]
LOCATION: [your city, e.g. "Jakarta, Indonesia"]

WORK / PROJECTS:
1. [Project name] — [one sentence description] — [link if any]
2. [Project name] — [one sentence description] — [link if any]
3. [Project name] — [one sentence description] — [link if any]

LINKS:
- Twitter/X: [your handle or URL]
- LinkedIn: [your URL]
- GitHub: [your URL]
- Email: [your email]

The page should have:
- A clean header with my name and role
- A short bio section
- A "Work" or "Projects" section showing my projects as cards
- A simple footer with my social links

Design: clean and minimal, easy to read, feels professional but personal. Light mode.

Use the existing Next.js app structure. Put everything in app/page.tsx. Use Tailwind CSS for styling.
```

---

## How to fill it in

Fill in each field with your real information. Skip any sections that don't apply — just remove them from the prompt.

---

## Expanding it later

- *"Add a 'Writing' section that links to my blog posts"*
- *"Add a contact form at the bottom"*
- *"Make this feel more like a personal website, less like a resume"*
- *"Add a dark mode toggle"*
