# Vibe Coding Starter

A practical guide for non-technical people who want to build and ship real things with AI.

No prior coding experience needed. Your job is to **direct**, not to write code.

---

## The mental shift

Building with AI in 2026 means directing a capable contractor:

- **You** describe what you want clearly
- **The AI** writes the code
- **You** review, test, and say "yes" or "change this"
- **Git** is a save button with history
- **Vercel** is a publish button

---

## What you'll build

By the end of these four guides, you'll have:

- A real Next.js project running on your computer
- A live URL you can share with anyone
- The confidence to keep building from here

---

## The four milestones

| # | Guide | What happens |
|---|---|---|
| 1 | [Setup](guides/01-setup.md) | Install the tools. One-time. Never again. |
| 2 | [Build](guides/02-build.md) | Tell the AI what you want. Watch it build. |
| 3 | [Save](guides/03-save.md) | Commit your work. Like saving a document, but smarter. |
| 4 | [Ship](guides/04-ship.md) | Put it live at a real URL. |

Start with [Guide 1 → Setup](guides/01-setup.md)
